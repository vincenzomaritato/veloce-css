Documentation

VeloceCSS is a modern, lightweight library of ready-to-use, cross-browser animations. Great for emphasis, home pages, sliders, and attention-guiding hints.

Installation

v1.0.0

Install with npm

bash
$ npm install veloce-css --save

Import via CDN

html
<link rel="stylesheet" href="..." />

Basic Usage

After installing, add the class vel-animated to an element, along with any of the animation names (don't forget the vel- prefix!):

html
<h1 class="vel-animated vel-bounce">An animated element</h1>

Tailwind CSS Integration

Plugin Available

Use VeloceCSS animations as Tailwind utility classes. Install the official plugin or configure manually.

Installation

bash
$ npm install @veloce/tailwind veloce-css

Configure Tailwind

tailwind.config.js
module.exports = {
  content: ['./src/**/*.{js,jsx,ts,tsx}'],
  plugins: [
    require('@veloce/tailwind'),
  ],
}

Import CSS

css
@import 'veloce-css/dist/veloce.min.css';

Usage

jsx
<div className="animate-vel-fadeInUp">
  Content
</div>

// All animations available as animate-vel-{name}
<div className="animate-vel-bounce animate-vel-pulse">
  Multiple animations
</div>

Extend Tailwind's theme to include VeloceCSS keyframes and animations.

tailwind.config.js
module.exports = {
  theme: {
    extend: {
      keyframes: {
        'vel-fadeInUp': {
          'from': {
            opacity: 0,
            transform: 'translate3d(0, 20px, 0)',
          },
          'to': {
            opacity: 1,
            transform: 'translate3d(0, 0, 0)',
          },
        },
        // Add more keyframes from VeloceCSS...
      },
      animation: {
        'vel-fadeInUp': 'velFadeInUp 1s ease-out',
        // Add more animations...
      },
    },
  },
}

Tip: For a complete setup, copy all keyframes from dist/veloce.css or use the @veloce/tailwind plugin for automatic integration.

CSS Custom Properties

VeloceCSS uses CSS variables to define duration, delay, and iterations. This makes it extremely flexible.

Global Change
css
:root {
  --vel-duration: 800ms;
  --vel-delay: 0.5s;
}
Local Change
css
.my-element {
  --vel-duration: 2s;
  --vel-repeat: infinite;
}

Utility Classes

Class Name Effect
.vel-delay-1s Adds 1s delay
.vel-delay-2s Adds 2s delay
.vel-fast Speed: 800ms
.vel-faster Speed: 500ms
.vel-slow Speed: 2s
.vel-slower Speed: 3s
.vel-infinite Infinite loop

Framework Integration

React • Vue

Use VeloceCSS with modern frameworks. Examples for React, Vue, and animation libraries.

React

Class-based (Simple)

jsx
function Card() {
  const [isVisible, setIsVisible] = useState(false);
  
  return (
    <div
      className=`vel-animated ${isVisible ? 'vel-fadeInUp' : ''}`
    >
      Content
    </div>
  );
}

With Framer Motion

jsx
import { motion } from 'framer-motion';

function Card() {
  return (
    <motion.div
      className="vel-animated vel-fadeInUp"
      initial={{ opacity: 0 }}
      animate={{ opacity: 1 }}
    >
      Content
    </motion.div>
  );
}

Vue.js

Class Binding

vue
<template>
  <div
    :class="['vel-animated', isVisible ? 'vel-fadeInUp' : '']"
  >
    Content
  </div>
</template>

<script setup>
import { ref } from 'vue';
const isVisible = ref(false);
</script>

With Transition Component

vue
<template>
  <Transition
    enter-active-class="vel-animated vel-fadeInUp"
    leave-active-class="vel-animated vel-fadeOut"
  >
    <div v-if="show">
      Content
    </div>
  </Transition>
</template>

💡 Pro Tips

  • Use useEffect (React) or onMounted (Vue) to trigger animations on mount
  • Combine with state management to control animation timing
  • Use CSS custom properties for dynamic duration: style={{'--vel-animate-duration': '2s'}}
  • For complex animations, consider Framer Motion (React) or Vue Transition Groups

Accessibility & Best Practices

Critical

Animations can greatly improve UX, but overuse can deteriorate it.

  • Don't animate root elements (html, body). Wrap your content instead.
  • Avoid infinite animations unless necessary (loading states).
  • Respect Reduced Motion: VeloceCSS automatically supports the prefers-reduced-motion media query. Never disable this feature.
